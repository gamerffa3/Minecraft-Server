# 🎮 Minecraft Server with Tailscale + GitHub Backup

## 🚀 Features
- ✅ Auto backup on server start/stop
- ✅ Real-time backup every 5 minutes
- ✅ GitHub automatic upload
- ✅ Tailscale VPN connection
- ✅ Auto delete old backups (keep last 7 days)
- ✅ 16GB RAM support
- ✅ All plugins auto-load

## 📋 Requirements
- Ubuntu/Debian or any Linux
- Java 21
- Git
- Tailscale account
